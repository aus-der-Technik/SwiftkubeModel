# Changelog

## 0.2.0

### New

- New marker protocols for the supported API actions
  - `ReadableResource`
  - `ListableResource`
  - `CreatableResource`
  - `ReplaceableResource`
  - `DeletableResource`
  - `CollectionDeletableResource`

### API Changes

- Renamed `GroupVersionKind` initializer argument from `init(forName:)` to `init(for:)`
